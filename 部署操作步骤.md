# 部署到Render - 详细操作步骤

## 第一步：准备GitHub仓库

### 1.1 创建GitHub仓库

1. 打开浏览器，访问 https://github.com
2. 点击右上角 "+" → "New repository"
3. 填写信息：
   - Repository name: `tcm-ai-assistant`（或你喜欢的名字）
   - Description: `中医AI助手Web版`
   - 选择 **Public**（公开）
   - **不要**勾选 "Add a README file"
4. 点击 "Create repository"

### 1.2 上传代码到GitHub

在 `中医AI助手Web版` 目录下打开命令行，执行：

```bash
# 初始化Git仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: 中医AI助手Web版"

# 添加远程仓库（替换成你的GitHub用户名）
git remote add origin https://github.com/你的用户名/tcm-ai-assistant.git

# 推送到GitHub
git branch -M main
git push -u origin main
```

**如果遇到问题**：
- 需要输入GitHub用户名和密码（或Personal Access Token）
- 如果没有配置Git，先运行：
  ```bash
  git config --global user.name "你的名字"
  git config --global user.email "你的邮箱"
  ```

## 第二步：部署到Render

### 2.1 登录Render

1. 访问 https://render.com
2. 点击右上角 "Get Started" 或 "Sign In"
3. 选择 "Sign in with GitHub"（推荐）
4. 授权Render访问你的GitHub账号

### 2.2 创建Web Service

1. 登录后，点击 "New +" → "Web Service"
2. 在 "Connect a repository" 页面：
   - 如果看到你的仓库，直接点击 "Connect"
   - 如果没看到，点击 "Configure account" 授权更多仓库
3. 选择 `tcm-ai-assistant` 仓库
4. 点击 "Connect"

### 2.3 配置Web Service

Render会自动检测 `render.yaml`，但你也可以手动确认：

**基本设置**：
- **Name**: `tcm-ai-assistant`（或自定义）
- **Region**: 选择 `Singapore`（离中国近）
- **Branch**: `main`
- **Root Directory**: 留空
- **Environment**: `Python 3`
- **Build Command**: `pip install -r requirements.txt`
- **Start Command**: `gunicorn app:app`

**计划选择**：
- 选择 **Free**（免费方案）

**环境变量**（可选）：
- 点击 "Advanced" → "Add Environment Variable"
- 可以添加：
  - `PYTHON_VERSION`: `3.9.0`
  - `FREE_DAILY_LIMIT`: `5`

### 2.4 开始部署

1. 检查所有配置无误
2. 点击 "Create Web Service"
3. 等待部署（3-5分钟）

## 常见问题

### Q1: 部署失败怎么办？

**查看错误日志**：
1. 在Render控制台点击 "Logs"
2. 查看红色错误信息
3. 常见错误：
   - `ModuleNotFoundError`: 检查 requirements.txt
   - `Port already in use`: 检查 app.py 端口配置
   - `Permission denied`: 检查文件权限

**解决方法**：
1. 修复本地代码
2. 重新推送到GitHub
3. Render会自动重新部署

### Q2: 访问很慢怎么办？

**原因**：
- 免费方案15分钟无活动会休眠
- 首次访问需要唤醒（30秒）

**解决**：
- 升级到付费方案（$7/月）
- 或使用定时任务保持活跃

### Q3: 数据丢失怎么办？

**原因**：
- 免费方案重启后数据会丢失

**解决**：
1. 定期备份数据库（下载 data/users.db）
2. 升级到付费方案（有持久化存储）
3. 使用外部数据库（PostgreSQL）

### Q4: API密钥安全吗？

**当前状态**：
- API密钥在 config.py 中（已上传GitHub）
- 仓库是Public（任何人可见）

**安全建议**：
1. 将仓库改为Private
2. 使用环境变量存储密钥
3. 定期更换密钥

### Q5: 如何自定义域名？

1. 在Render控制台进入你的Web Service
2. 点击 "Settings" → "Custom Domain"
3. 添加你的域名
4. 在域名DNS设置中添加CNAME记录
