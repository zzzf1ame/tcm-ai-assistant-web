## 🚀 部署步骤

### 1. 准备GitHub仓库
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/用户名/tcm-ai-assistant.git
git push -u origin main
```

### 2. 在Render创建Web Service
1. 登录 https://render.com
2. New + → Web Service
3. Connect GitHub仓库
4. 选择 tcm-ai-assistant
5. 配置自动检测（render.yaml）
6. 点击 Create Web Service

### 3. 等待部署完成
- 构建时间: 2-3分钟
- 启动时间: 30秒-1分钟
- 总时间: 3-5分钟

### 4. 访问应用
- URL: https://tcm-ai-assistant.onrender.com
- 首次访问可能需要30秒唤醒
