# 使用GitHub Desktop上传（最简单）

## 为什么推荐GitHub Desktop？

- ✅ 图形界面，不需要命令行
- ✅ 自动处理认证，不需要Token
- ✅ 一键上传，简单快速

---

## 步骤1：下载安装GitHub Desktop

1. 访问：https://desktop.github.com/
2. 点击 "Download for Windows"
3. 下载完成后双击安装
4. 安装完成后打开GitHub Desktop

---

## 步骤2：登录GitHub账号

1. 打开GitHub Desktop
2. 点击 "Sign in to GitHub.com"
3. 在浏览器中登录你的GitHub账号
4. 授权GitHub Desktop访问

---

## 步骤3：添加本地仓库

1. 在GitHub Desktop中，点击 "File" → "Add local repository"
2. 点击 "Choose..." 按钮
3. 选择文件夹：`中医AI助手Web版`
4. 点击 "Add repository"

如果提示 "This directory does not appear to be a Git repository"：
- 点击 "create a repository"
- 点击 "Create repository"

---

## 步骤4：发布到GitHub

1. 在GitHub Desktop中，点击 "Publish repository"
2. 确认信息：
   - Name: `tcm-ai-assistant`
   - Description: `中医AI助手Web版`
   - **取消勾选** "Keep this code private"（保持公开）
3. 点击 "Publish repository"
4. 等待上传完成（1-2分钟）

---

## 步骤5：确认上传成功

1. 访问：https://github.com/zzzflame/tcm-ai-assistant
2. 刷新页面
3. 应该能看到所有文件

---

## 完成！

上传成功后，告诉我，我帮你继续部署到Render！

---

## 如果已经有仓库

如果你已经在GitHub创建了 `tcm-ai-assistant` 仓库：

1. 在GitHub Desktop中，点击 "File" → "Clone repository"
2. 选择 "GitHub.com" 标签
3. 找到 `zzzflame/tcm-ai-assistant`
4. 选择保存位置
5. 点击 "Clone"
6. 然后把 `中医AI助手Web版` 文件夹里的所有文件复制到克隆的文件夹
7. 在GitHub Desktop中会自动检测到更改
8. 输入提交信息："Initial commit"
9. 点击 "Commit to main"
10. 点击 "Push origin"

---
