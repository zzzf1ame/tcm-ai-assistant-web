# 本地测试指南

## 安装依赖

```bash
pip install Flask Flask-CORS bcrypt requests
```

## 运行应用

```bash
python app.py
```

应用会在 http://localhost:5000 启动

## 测试流程

### 1. 注册账号

1. 访问 http://localhost:5000
2. 点击"注册"标签
3. 填写信息：
   - 用户名：test123
   - 密码：123456
   - 手机号：13800138000
4. 点击"注册"

### 2. 完善档案

注册成功后会自动跳转到对话页面，但需要先完善档案：

1. 点击左侧"个人档案"
2. 填写信息：
   - 年龄：30
   - 性别：男
   - 体质：气虚质
   - 地区：北京
3. 点击"保存档案"

### 3. 开始对话

1. 点击左侧"对话"
2. 输入问题，如："我最近总是感觉疲劳，怎么办？"
3. 点击"发送"或按 Ctrl+Enter
4. 等待AI回复

### 4. 测试使用限制

连续发送6条消息，第6条应该提示"今日使用次数已达上限"

### 5. 测试防刷机制

尝试用同一IP注册4个账号，第4个应该提示"该IP今日注册次数已达上限"

## 常见问题

### 1. 端口被占用

修改 app.py 最后一行：
```python
app.run(host='0.0.0.0', port=5001, debug=True)
```

### 2. 数据库锁定

关闭所有使用数据库的程序，删除 data/users.db 重新运行

### 3. AI不回复

检查 config.py 中的API密钥是否正确

### 4. 样式不显示

确认 static/css/style.css 文件存在

## 性能测试

### 并发测试

使用 Apache Bench:
```bash
ab -n 100 -c 10 http://localhost:5000/
```

### 响应时间测试

使用浏览器开发者工具的Network标签查看各个请求的响应时间

## 调试技巧

### 1. 查看Flask日志

运行时会在终端显示所有请求日志

### 2. 浏览器控制台

按F12打开开发者工具，查看Console中的JavaScript错误

### 3. 网络请求

在Network标签查看所有API请求和响应

### 4. 数据库查询

在代码中添加打印语句：
```python
print(f"User ID: {user_id}")
print(f"Profile: {profile}")
```
