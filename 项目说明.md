# 中医AI助手Web版 - 项目说明

## 项目概述

这是一个基于Flask的Web应用，提供中医健康咨询服务。用户可以注册账号，填写个人档案，与AI进行中医养生对话。

## 核心特性

### 1. 用户系统
- 注册/登录/登出
- 密码bcrypt加密
- Session管理
- 防刷机制（IP限制）

### 2. 个人档案
- 年龄、性别、体质、地区
- 个性化AI建议
- 档案修改

### 3. AI对话
- 基于通义千问Qwen3-max
- 知识库增强（82条中医知识）
- 对话历史记忆（最近5条）
- Markdown渲染

### 4. 使用限制
- 免费用户每天5次
- 实时显示剩余次数
- 超限提示

### 5. 响应式设计
- 支持手机、平板、电脑
- 自适应布局
- 移动端优化

## 数据库设计

### users表
```sql
id              INTEGER PRIMARY KEY
username        TEXT UNIQUE NOT NULL
password_hash   TEXT NOT NULL
phone           TEXT UNIQUE NOT NULL
created_at      TIMESTAMP
last_login      TIMESTAMP
is_vip          BOOLEAN DEFAULT 0
vip_expire      TIMESTAMP
```

### profiles表
```sql
id              INTEGER PRIMARY KEY
user_id         INTEGER NOT NULL (FK)
age             INTEGER
gender          TEXT
constitution    TEXT
location        TEXT
created_at      TIMESTAMP
updated_at      TIMESTAMP
```

### conversations表
```sql
id              INTEGER PRIMARY KEY
user_id         INTEGER NOT NULL (FK)
user_message    TEXT NOT NULL
ai_response     TEXT NOT NULL
timestamp       TIMESTAMP
```

### usage_log表
```sql
id              INTEGER PRIMARY KEY
user_id         INTEGER NOT NULL (FK)
date            TEXT NOT NULL
count           INTEGER DEFAULT 1
ip              TEXT
user_agent      TEXT
UNIQUE(user_id, date)
```

### registration_log表
```sql
id              INTEGER PRIMARY KEY
ip              TEXT NOT NULL
date            TEXT NOT NULL
count           INTEGER DEFAULT 1
UNIQUE(ip, date)
```

## API接口

### 认证相关
- `POST /api/register` - 用户注册
- `POST /api/login` - 用户登录
- `POST /api/logout` - 用户登出
- `GET /api/check_auth` - 检查登录状态

### 用户相关
- `GET /api/profile` - 获取个人档案
- `POST /api/profile` - 保存/更新档案
- `GET /api/usage` - 获取今日使用情况

### 对话相关
- `POST /api/chat` - 发送消息
- `GET /api/conversations` - 获取对话历史

## 安全机制

### 1. 密码安全
- bcrypt加密存储
- 最小长度6位
- 不明文传输

### 2. Session安全
- 随机密钥
- 7天有效期
- HttpOnly Cookie

### 3. 防刷机制
- IP注册限制（3次/天）
- 使用次数限制（5次/天）
- 设备指纹记录

### 4. 输入验证
- 用户名长度限制
- 手机号格式验证
- 年龄范围检查
- SQL注入防护

## 常见问题

### Q: 为什么选择Web版而不是桌面版？
A: 手机用户更多，Web版无需安装，跨平台，易于分享。

### Q: 免费方案够用吗？
A: 对于初期用户（200-500人）完全够用。

### Q: 数据会丢失吗？
A: Render免费方案重启后数据会丢失，建议定期备份或升级付费方案。

### Q: 如何升级到付费版？
A: 在Render控制台选择Starter方案（$7/月），获得持久化存储。

### Q: 可以自定义域名吗？
A: 可以，在Render设置中添加自定义域名。
