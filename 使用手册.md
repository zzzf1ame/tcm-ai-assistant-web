## 用户指南

### 1. 注册账号

#### 步骤
1. 访问首页
2. 点击"注册"标签
3. 填写信息：
   - **用户名**: 3-20个字符，字母数字组合
   - **密码**: 至少6个字符
   - **手机号**: 11位数字
4. 点击"注册"按钮

#### 注意事项
- 用户名不能重复
- 手机号不能重复
- 同一IP每天最多注册3个账号

### 2. 登录系统

#### 步骤
1. 在首页输入用户名和密码
2. 点击"登录"按钮
3. 登录成功后自动跳转到对话页面

#### 记住登录
- 登录状态保持7天
- 关闭浏览器后仍然有效
- 可以手动退出登录

### 3. 完善个人档案

#### 为什么要填写档案？
个人档案帮助AI了解你的体质特点，给出更精准的个性化建议。

#### 步骤
1. 点击左侧"个人档案"
2. 填写信息：
   - **年龄**: 1-120岁
   - **性别**: 男/女
   - **体质**: 选择你的体质类型
   - **地区**: 你所在的城市
3. 点击"保存档案"

#### 体质类型说明

| 体质 | 特点 | 调理重点 |
|------|------|----------|
| 平和质 | 身体健康，精力充沛 | 保持现状 |
| 气虚质 | 容易疲劳，气短 | 补气 |
| 阳虚质 | 怕冷，手脚冰凉 | 温阳 |
| 阴虚质 | 口干，手心热 | 滋阴 |
| 痰湿质 | 体型肥胖，容易困倦 | 祛湿 |
| 湿热质 | 面部油腻，口苦 | 清热 |
| 血瘀质 | 面色晦暗，易长斑 | 活血 |
| 气郁质 | 情绪低落，易焦虑 | 疏肝 |
| 特禀质 | 过敏体质 | 调理免疫 |

### 4. 开始对话

#### 步骤
1. 点击左侧"对话"
2. 在输入框输入问题
3. 点击"发送"或按 Ctrl+Enter
4. 等待AI回复

#### 提问技巧

**好的提问示例：**
- "我最近总是感觉疲劳，怎么办？"
- "气虚体质适合吃什么食物？"
- "有什么穴位可以缓解头痛？"
- "冬天如何养生？"

**不好的提问示例：**
- "你好"（太简单）
- "帮我看病"（AI不能诊断疾病）
- "开个药方"（AI不能开处方）

#### AI回复特点
- 基于你的体质给出个性化建议
- 提供具体的食疗方案
- 推荐穴位按摩方法
- 给出生活调理建议
- 支持Markdown格式（标题、列表、加粗等）

### 5. 查看对话历史

- 每次进入对话页面会显示最近10条对话
- AI会记住最近5条对话的上下文
- 可以点击"清空"按钮清空当前显示

### 6. 使用限制

#### 免费用户
- 每天5次对话
- 每天0点重置
- 超限后提示"今日使用次数已达上限"

#### 查看剩余次数
- 左侧导航栏显示"今日使用: X/5"
- 实时更新

### 7. 退出登录

点击左侧"退出登录"按钮

## 管理员指南

### 配置文件

#### config.py

```python
# API配置
API_PROVIDER = "qwen"  # 使用通义千问
QWEN_API_KEY = "你的API密钥"
MODEL = "qwen3-max-2026-01-23"

# 使用限制
FREE_DAILY_LIMIT = 5  # 每天5次
```

### 数据库管理

#### 查看数据库

```bash
sqlite3 data/users.db
```

#### 常用SQL命令

```sql
-- 查看所有用户
SELECT id, username, phone, created_at FROM users;

-- 查看今日使用情况
SELECT u.username, ul.count, ul.date
FROM usage_log ul
JOIN users u ON ul.user_id = u.id
WHERE ul.date = date('now');

-- 查看注册统计
SELECT date, COUNT(*) as count
FROM users
GROUP BY date(created_at)
ORDER BY date DESC;

-- 重置某用户今日使用次数
DELETE FROM usage_log
WHERE user_id = 1 AND date = date('now');

-- 删除用户（慎用）
DELETE FROM users WHERE id = 1;
```

### 监控指标

#### 关注指标
- 每日活跃用户数
- 对话次数
- API调用成功率
- 响应时间
- 错误率

#### 查询示例

```sql
-- 今日活跃用户
SELECT COUNT(DISTINCT user_id) as active_users
FROM usage_log
WHERE date = date('now');

-- 今日对话总数
SELECT SUM(count) as total_chats
FROM usage_log
WHERE date = date('now');

-- 用户留存率
SELECT
  date(created_at) as register_date,
  COUNT(*) as new_users,
  (SELECT COUNT(DISTINCT user_id)
   FROM usage_log
   WHERE date > date(u.created_at)) as retained_users
FROM users u
GROUP BY date(created_at);
```

## 常见问题

### 用户问题

#### Q1: 忘记密码怎么办？
A: 目前没有找回密码功能，请联系管理员重置。

#### Q2: 可以修改用户名吗？
A: 目前不支持修改用户名。

#### Q3: 为什么AI不回复？
A: 可能原因：
- 网络问题
- API额度用完
- 服务器错误
请稍后重试或联系管理员。

#### Q4: 对话历史会保存多久？
A: 永久保存（除非管理员删除）。

#### Q5: 可以导出对话记录吗？
A: 目前不支持，后续版本会添加。

#### Q6: 为什么提示"今日使用次数已达上限"？
A: 免费用户每天限5次，请明天再来或联系管理员升级会员。

#### Q7: 手机上可以用吗？
A: 可以，界面已适配移动设备。

#### Q8: AI的建议可靠吗？
A: AI建议仅供参考，不能替代医生诊断。遇到严重症状请及时就医。

### 技术问题

#### Q1: 如何本地运行？
A: 参考"快速开始"章节。

#### Q2: 如何部署到服务器？
A: 参考"部署指南.md"。

#### Q3: 数据库在哪里？
A: `data/users.db`

#### Q4: 如何备份数据？
A: 复制 `data/users.db` 文件。

#### Q5: 如何修改使用限制？
A: 修改 `config.py` 中的 `FREE_DAILY_LIMIT`。

#### Q7: 支持其他AI模型吗？
A: 目前支持通义千问和DeepSeek，修改 `API_PROVIDER` 切换。

A: 修改相应的Python文件和HTML模板。
